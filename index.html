Setup is simple.

Five easy steps to get you flowing.

1

Download the Extension

Get the latest version of Study Flow directly from our repository. It comes as a secure ZIP file ready for manual installation.

2

Unzip the File

Locate the downloaded file and extract it. You should see a folder containing the manifest and source files.

3

Enable Developer Mode

Navigate to chrome://extensions in your browser bar. Toggle the Developer mode switch in the top right corner.

4

Drag & Drop to Install

Simply drag the extracted folder directly into the extensions window. Alternatively, click "Load unpacked" and select the folder.

5

You're All Set!

Study Flow is now active. Pin it to your toolbar and start upgrading your productivity today.
